# CompadreFood

Loja de delivery de bebidas (Next.js 16 / React 19 / Tailwind v4), deploy na Vercel.

- Checkout PIX + Cartão via Pagou.ai (v2)
- Confirmação de pedido por e-mail (Resend) + webhook + KV (Upstash)
- E-mail de carrinho abandonado (QStash)
- Painel admin em `/admin`

Veja `.env.example` para as variáveis de ambiente necessárias.
